# Easy Level Tester
Designing levels for games can be hard, but it shouldn't be hard to test them! This tool can be used to quickly generate a game around your level, allowing rapid prototyping.

<img width="100%" height="100%" alt="Untitled-1" src="https://github.com/user-attachments/assets/7f5254ee-3e14-4371-bdaf-914d69d295b2" />


This tool supports:
- Point and click interaction
- Navigation that adheres to walkable or non navigable terrain
- Weapon or Health Powerups
- Enemy AI
- Static or patrolling enemies
- Combat
- And more!
